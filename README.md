This is a [Next.js](https://nextjs.org/) project 

## Установка 


```bash

git clone https://github.com/gudkovWay/tvoe

cd tvoe

npm install
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
